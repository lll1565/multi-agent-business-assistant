# Web Search 工作流

按以下流程做公网搜索（最多 2 轮）：

1. **第 1 轮**：用 `search_web(query=用户原始问题, max_results=5)` 搜一次
2. **判断结果**：
   - 有 ≥ 3 条相关结果 → 直接整理回答
   - 结果太少或都不相关 → 换关键词再搜一次
3. **第 2 轮**：用更具体的关键词（包含专有名词 / 英文名 / 时间范围）再搜
4. **仍然没结果** → 老实告诉用户"未找到相关结果"，建议换个问法

## 整理回答时

- 每条要点必须附上来源 URL
- 时效性内容注明"截至 YYYY-MM"或"信息来源时间 X"
- 冲突信息要并列呈现，不要只挑一个
